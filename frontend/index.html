ShareBite is a cloud-based food redistribution platform designed to connect households, restaurants, supermarkets and other food donors with people and communities in need through a scalable AWS serverless architecture.

How ShareBite Works

ShareBite provides a simple process that allows surplus food to be listed, stored securely in the cloud and made visible to people who can collect it.

1

Donor Lists Food

The donor enters the food name, quantity, collection location, expiry time and contact information.

2

Cloud Processing

API Gateway sends the request to AWS Lambda, which validates the information and stores it in Amazon DynamoDB.

3

Recipient Finds Food

Available food listings are retrieved live so recipients can view the location, deadline and donor contact details.

About ShareBite

ShareBite is a university cloud computing project created to demonstrate how serverless technology can support a practical food redistribution service. The platform allows donors to publish surplus food and enables recipients to view available listings.

The current implementation uses Amazon API Gateway for the web API, AWS Lambda for backend processing, Amazon DynamoDB for food listing storage, Amazon SNS for notifications, Amazon CloudWatch for monitoring, Amazon S3 for web hosting support and AWS CloudFormation for infrastructure deployment.

The project aligns with United Nations Sustainable Development Goal 2: Zero Hunger, while also encouraging responsible food use and reduced waste.
